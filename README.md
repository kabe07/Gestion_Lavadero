Programa para gestion de un lavadero de autos desarrollado en c# y winforms.
Permite agregar clientes y vehiculos, luego seleccionarlos para crear turnos.
![Pantalla principal](docs/Principal.jpg)

## Funcionalidades

- Alta de clientes
- Asociación de vehículos a un cliente
- Registro de distintos tipos de vehículos (Auto, Moto, SUV, Pickup)
- Gestión de servicios (Lavado, Encerado, Limpieza de tapizados)
- Asignación de turnos con fecha y hora
- Cálculo automático del precio según tipo de vehículo
- Visualización de turnos ordenados cronológicamente
- Selección de cliente para ver sus vehículos asociados
## Modelo del Sistema

- Cliente
- Vehiculo 
- Servicio
- Turno
- SistemaLavadero (gestiona las operaciones generales)
## Tecnologías utilizadas

- C#
- .NET
- Windows Forms (WinForms)
- Programación orientada a objetos
- Colecciones genéricas (List y Dictionary)
- Serialización de datos en archivos

## Cómo ejecutar

1. Clonar el repositorio
2. Abrir el archivo GestionLavadero.sln en Visual Studio 2022
3. Compilar la solución (Ctrl + Shift + B)
4. Ejecutar el programa (F5)
